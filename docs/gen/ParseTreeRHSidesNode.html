Class: ParseTreeRHSidesNode

ParseTreeRHSidesNode

Internal class for representing all parse trees with a given starting nonterminal in a subderivation
ParseTreeRHSidesNode is a recursively defined data structure of form {lhs: string, rhsides: Array.<string|Array.>} where lhs is the common starting nonterminal; rhsides represents all possible immediate children parse trees resulting from one application of a production rule (rhs is a root in rhsides exactly when lhs -> rhs can be used at this step in one of the possible derivations)
